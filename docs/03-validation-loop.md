# 验收清单

这个清单适合检查数据可视化管理平台 demo 是否真的能讲通。

## 状态联动

- 切换场景后，关系图有明确变化。
- 告警列表跟当前场景有关。
- 健康度分数和规则项能解释变化。
- 数据校验结果会影响报表信心。

## 数据可信度

- 每类源数据都有行数。
- 匹配、缺失、异常都有单独展示。
- 缺失数据不会被静默吞掉。
- 页面能解释当前结果来自哪些输入表和数据库统计。

## 演示体验

- 第一屏能看懂系统在做什么。
- 关键指标不需要说明书。
- 图、表、规则之间有关系。
- 空状态有明确提示。
- 移动端不会挤到读不清。

## 数据边界检查

- 示例表字段足够说明结构。
- 示例数据能跑通导入和导出。
- `output/` 这类生成结果不提交。
- 配置通过环境变量或本地文件读取。
- 仓库里不放 `.env`。
