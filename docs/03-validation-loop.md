# 验收清单

这个清单适合检查 To B dashboard demo 是否真的能讲通。

## 状态联动

- 切换场景后，关系图有明确变化。
- 告警列表跟当前场景有关。
- 健康度分数和规则项能解释变化。
- 数据校验结果会影响报表信心。

## 数据可信度

- 每类源数据都有行数。
- 匹配、缺失、异常都有单独展示。
- 缺失数据不会被静默吞掉。
- 页面能解释当前结果来自哪些 mock 结构。

## 演示体验

- 第一屏能看懂系统在做什么。
- 关键指标不需要说明书。
- 图、表、规则之间有关系。
- 空状态有明确提示。
- 移动端不会挤到读不清。

## 公开检查

- 没有真实业务数据。
- 没有真实文件名。
- 没有真实截图。
- 没有真实接口路径。
- 没有内部术语。
- 没有任何密钥或 `.env`。
